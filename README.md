# Rastgele Sayı Üreteci

Bu proje, **Collatz dizisi** kullanılarak geliştirilmiş
basit bir **sözde rastgele sayı üreteci (PRNG)** örneğidir.

Üreteç deterministiktir; aynı başlangıç değeri (seed)
kullanıldığında aynı çıktı üretilir.
Kriptografik amaçlar için uygun değildir.

## Çalıştırma

```bash
python collatz_rng2.py
